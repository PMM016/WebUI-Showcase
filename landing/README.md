A collection of **20 modern, fully responsive, and highly customizable** single-page landing page templates built with love for my GitHub portfolio.

Each template includes:
- **Theme Toggler** (Light/Dark)
- **Advanced Color Manipulator** (Settings popup with color pickers + Reset button)
- Beautiful animations (AOS + custom SVG + Squiggle effects)
- Swiper.js carousels, Vanilla Counters, and smooth interactions

---

## ✨ Features (Common to All Templates)

- **Fonts**: Nunito (Headings), Ubuntu (Subheadings), Open Sans (Body)
- **CDNs**: Bootstrap 5, Font Awesome 6, Bootstrap Icons, AOS, Swiper.js, Vanilla Counter
- **Theme Toggler** in Navbar
- **Settings Popup** (⚙️) with:
  - Multiple color pickers for `--primary`, `--secondary`, `--accent`
  - Real-time color updates
  - Reset to default button
  - localStorage support
- Custom SVG animations & Squiggle hover links
- Fully responsive & mobile-first
- Clean, well-commented code

## 🚀 How to Use Any Template

1. Navigate to the desired folder
2. Open `index.html` in your browser
3. Click the **⚙️ Settings** button (bottom right) to customize colors
4. Use the **☀️/☽️** theme toggler in the navbar
